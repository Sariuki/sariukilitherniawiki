# Kingdom_of_Mordhul

Aquí yacen las crónicas del Reino de Mordhul, un dominio forjado en la sombra y el poder ancestral. Sus archivos no guardan [[Lithernia/12_Atlas/Mapas\|Mapas]] ni tratados, sino las [[Lithernia/09_Personajes/Leyendas\|Leyendas]] de una dinastía caída. Desde [[Lithernia/02_Reinos/Kingdom_of_Mordhul/Aelithor\|Aelithor]] hasta [[Lithernia/02_Reinos/Kingdom_of_Mordhul/Zelindra\|Zelindra]], cada nombre es el eco de una gloria corrompida, un pilar en la historia de un trono oscuro cuyas [[Lithernia/09_Personajes/Leyendas\|Leyendas]] aún resuenan con una majestad terrible.
